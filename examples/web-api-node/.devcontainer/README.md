## Quick Start

This development environment uses Docker Compose to orchestrate multiple services.

**Starting the environment:**

1. Open this folder in VS Code
2. When prompted, click "Reopen in Container"
3. Wait for the container to build and services to start
4. Your development environment is ready!

## Environment Variables

This project uses environment variables for configuration. A template is provided:

```bash
cp .env.example .env
```

Then edit `.env` to customize for your environment.

**Important:** The `.env` file is git-ignored and should not be committed.
